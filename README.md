# event space

## Prerequisites
* Java 1.8
* Maven 3.5

## Installing
* Run this in your terminal:
```sh
git clone https://github.com/cecemei/event-space.git
cd event-space
mvn clean install
java -jar target/meet-up-open-events-0.1.1-SNAPSHOT.jar
```
